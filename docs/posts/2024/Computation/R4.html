1 Expression とは

オブジェクト志向言語ではコード自体もオブジェクトであり，これを R では Expression と呼ぶ．

１つのクラスからなるわけではなく，call, symbol, constant, pairlist の４つの型からなる．1

1.1 コンストラクタ

次のような操作ができる2

rlang::expr がコンストラクタである：

library(rlang)
z <- rlang::expr(y <- x*10)
z
y <- x * 10

expression オブジェクトは base::eval() で評価できる：

x <- 4
eval(z)
y
[1] 40

1.2 参照方法

expression には list のようにアクセス可能である：3

f <- expr(f(x = 1, y = 2))

# Add a new argument
f$z <- 3
f
f(x = 1, y = 2, z = 3)
#> f(x = 1, y = 2, z = 3)

# Or remove an argument:
f[[2]] <- NULL
f
f(y = 2, z = 3)
#> f(y = 2, z = 3)

2 メタプログラミング

capture_it <- function(x) {
  expr(x)
}
capture_it(a + b + c)
x
capture_it <- function(x) {
  enexpr(x)
}
capture_it(a + b + c)
a + b + c
